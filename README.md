## 1. 프로젝트 소개
- 프로젝트 이름 : iamFolio
- 프로젝트 내용 : 포트폴리오 관리를 위한 게시판 웹서비스 

### 개발 기간
> 2023.11.14 - 2024.12.04 <br>

### 인원 구성
> 유민준(1인)

## 2. 주요기능
- 로그인 및 회원가입 
- 파일 업로드 및 다운로드 
- 게시물 등록,수정,삭제
- 보안 : SpringSecurity

- 사용자는 로그인 후 서비스를 이용가능하며, 비회원이 게시판에 접근시 로그인 페이지로 이동합니다. <br>

- 빈칸없이 모두 입력하여 회원가입을 진행해야며, 비밀번호는 최소 4자, 최대 12자를 입력해야합니다. <br>
